Información del proyecto:

  • Este es un proyecto personal realizado para prácticar html,javascript y css.
  • También realizo varias consultas a una base datos personal para conseguir los datos que se representan en las gráficas. Para ello utilizo el framework Express.js y una base de datos MySQL.
  • Los datos se recogen gracias a la información que proporciona el bot de telegram que he realizado conjuntamente a este proyecto. El usuario envía su ubicación al bot de telegram y este hace una request a una API REST de la página oficial de carburantes del Gobierno de España. Después se procesan los datos mediante un filtrado que devolverá las gasolineras más cercanas dependiendo de la ubicación del usuario y del precio del carburante seleccionado. Bot de Telegram .
  • Frameworks Python: Telegram, Requests, Json, MySQL, Datetime y Geopy.
  • Frameworks JavaScript: Chart, Cors, MySQL2 y Express.
Commands 
                
                    select message as data,count(message) as count 
                    from logs 
                    where message in ('start','f95','diesel') 
                    group by message;
CCAA 
                
                    select name as data, count(name) as count 
                    from ccaa_register as r 
                    inner join ccaa as c 
                    on c.id_ccaa=r.location_id 
                    group by name;
Most Used 
                
                    select date(msg_date) as data,count(msg_date) as count 
                    from logs
                    where message in ('f95','diesel')
                    group by date(msg_date) 
                    order by count(msg_date) desc 
                    limit 6;
